## 📖 Sobre

Exercício individual de simulação de atendimento no **Node.js REPL** (Read-Eval-Print Loop). O projeto foi desenvolvido para praticar comandos JavaScript diretamente no terminal, sem necessidade de navegador.

## 🧠 Conceitos Praticados

- **Variáveis:** declaração e escopo
- **Condicionais:** `if`/`else` para lógica de atendimento
- **Laços:** `while`, `for`, `forEach` para fluxos repetitivos
- **Funções:** declaração, parâmetros e retorno
- **Console API:** `console.log()`, `console.table()`, `console.error()`
